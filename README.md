## 🚀 GET HYDRATED IN A FLASH! (Installation) 🚀

This is the easy part. Seriously. Forget complex commands.

### The 1-Click HYDRATION STATION Install (Recommended!)

1.  **GO TO THE APP** in your browser (Chrome, Edge, etc.).
2.  **LOOK FOR THE INSTALL ICON** in the address bar. It looks like a little screen with a down arrow.
3.  **CLICK IT. BOOM.** You're done. It's now an app on your computer.

<br>

<details>
<summary><strong>👨‍💻 For the Code Wizards (Developer Setup)</strong></summary>

If you wanna get your hands dirty with the code...

1.  **Clone this bad boy:**
    ```bash
    git clone <your-repository-url>
    cd hydroflow
    ```

2.  **Install the magic spells (dependencies):**
    ```bash
    npm install  # or yarn install
    ```

3.  **UNLEASH THE BEAST (start the server):**
    ```bash
    npm start  # or yarn start
    ```
Now go to `http://localhost:3000` and witness the glory.

## 🎮 HOW TO BECOME A HYDRO-HOMIE (Usage) 🎮

1.  **Set Your Timer:** Slide that "Reminder Interval" to your perfect nagging frequency.
2.  **Pick Your Anthem:** Choose a notification sound that screams "DRINK WATER NOW!"
3.  **SMASH THE PLAY BUTTON:** Let the countdown begin.
4.  **Obey The App:** When it yells at you, drink that sweet, sweet water.
5.  **Claim Your Victory:** Click "I drank water" to reset the timer and log your triumph.
6.  **Don't Kill the Vibe:** MINIMIZE the app, don't close it, to keep the reminders flowing.

## 🔧 TWEAK THE TAPS & MAKE IT YOURS! (Configuration) 🔧

Feeling adventurous? Dive into `src/App.tsx` and mess with stuff!

*   `DEFAULT_INTERVAL`: Change the starting reminder time.
*   `SOUND_OPTIONS`: Add your own custom sounds! Find an MP3, host it, and add it to the list.
*   `HYDRATION_IMAGES`: Swap out the reminder images for your favorite memes. We won't judge.
